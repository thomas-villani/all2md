# About This Site

This website is a demonstration of the **all2md** library's capabilities for building markdown-powered web applications.

## About all2md

all2md is a comprehensive Python library for bidirectional document conversion. It supports:

- **200+ file formats** including PDF, Word, PowerPoint, HTML, and more
- **Markdown-first approach** for easy content management
- **AST-based processing** for powerful document transformation
- **Extensible plugin system** for custom formats

## About This Demo

This Flask application demonstrates:

1. **Frontmatter Parsing** - YAML metadata extraction from markdown files
2. **Dynamic Content** - Automatic post listing and sorting
3. **URL Routing** - Clean URLs with slug support
4. **HTML Rendering** - Beautiful, responsive output

## Technical Details

The site is built with:
- **Python 3.12+**
- **Flask** web framework
- **all2md** for markdown processing

All source code is available in the `examples/` directory of the all2md repository.
